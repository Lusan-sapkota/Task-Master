## ğŸš€ Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Lusan-sapkota/Task-Master.git
   cd Task-Master
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run Locally**
   ```bash
   ionic serve
   ```

4. **Deploy to Android**
   ```bash
   npx cap copy android
   npx cap open android
   ```
